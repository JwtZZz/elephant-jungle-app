# 什么是稳定币

稳定币是锚定法币或其他资产价格的加密资产，主要用途包括链上支付、交易对定价、跨平台转移和 DeFi 抵押。稳定币的目标是在高波动的加密市场中提供价格稳定性，让用户可以在不退出链上生态的情况下规避波动风险。

# 法币储备型稳定币

法币储备型稳定币（如 USDT、USDC）由现金或短期国债等传统资产支持，通常按 1:1 锚定美元。发行方声称每发行一枚稳定币就持有一美元的储备资产。这类稳定币的核心信任基础是发行方的透明度和审计质量。

# 加密抵押型稳定币

加密抵押型稳定币（如 DAI）依赖链上超额抵押和清算机制维持锚定。用户存入 ETH 等波动性资产作为抵押品，按一定抵押率铸造稳定币。如果抵押品价值下跌到清算线以下，协议会自动拍卖抵押品来偿还债务。这类稳定币的核心风险是抵押品价格剧烈波动和清算机制的可靠性。

# 算法稳定型稳定币

算法稳定型稳定币通过供给调节或市场激励维持价格，不依赖足额抵押。典型的机制包括 seigniorage（铸币税）模型和 rebase 机制。这类稳定币历史上发生过多次严重脱锚事件（如 UST/Luna 崩盘），风险最高。

# 稳定币的核心风险

稳定币的核心风险包括储备透明度不足、赎回机制受限、监管政策变化、抵押品价格剧烈波动、流动性危机下的死亡螺旋，以及智能合约漏洞。投资者和用户应了解不同稳定币的风险模型差异。
