# 钱包是什么

加密钱包不是"存钱"的工具，而是管理私钥的工具。代币始终存储在区块链上，钱包的作用是生成和保管私钥、签名交易、与 DApp 交互。谁控制私钥，谁就控制对应地址的资产。理解这一点是理解加密货币所有权的基础。

# 钱包类型

热钱包始终联网（如 MetaMask 浏览器插件、手机钱包 App），方便但攻击面大。冷钱包离线存储私钥（如 Ledger、Trezor 硬件钱包），安全性高但使用不便。纸钱包将私钥打印在纸上，适合长期持有但不适合频繁操作。多签钱包需要多方签名才能执行交易，适合团队和企业资金管理。

# 私钥与助记词

私钥是一个随机生成的 256 位数字，用十六进制表示。助记词（Mnemonic）是 12 或 24 个英文单词，可以确定性派生出一组私钥（BIP-39 标准）。备份助记词就等于备份了整个钱包的所有地址和资产。任何人获取了你的助记词就能完全控制你的资产。

# 托管方案

自托管（Self-custody）意味着用户自己管理私钥，完全控制但也完全负责——丢失助记词就意味着永远无法恢复资产。托管服务（如交易所内置钱包、机构托管商）替用户管理私钥，操作简便但引入了对手方风险，如交易所破产、被黑或冻结账户。MPC（多方计算）钱包将私钥分片存储在多个位置，是一种兼顾安全性和便利性的新方案。

# 安全最佳实践

永远不要在网上分享助记词或私钥。使用硬件钱包存储大额资产。定期检查 DApp 授权（token approval），撤销不用的授权。小心钓鱼网站和假冒钱包应用。交易前核对接收地址的前后几位字符，防止剪贴板劫持恶意软件篡改地址。
