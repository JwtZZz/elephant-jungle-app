# 工作量证明（PoW）

工作量证明依赖算力竞争。矿工通过消耗电力和硬件资源（ASIC 矿机）来争夺出块权。谁先计算出符合难度要求（前面有一定数量的零）的哈希值，谁就获得出块权和区块奖励。比特币是 PoW 的代表网络。PoW 的安全性建立在物理资源成本之上——攻击者需要购买和控制全网 51% 以上的算力，成本极高。

# 权益证明（PoS）

权益证明依赖质押资产和验证规则。验证者需要锁定一定数量的原生代币（如以太坊要求 32 ETH）作为质押，网络随机选择验证者来提议和验证区块。行为不当的验证者会被罚没质押金（Slashing）。以太坊是目前最大的 PoS 网络。

# 能耗对比

PoW 的最大争议是能耗。比特币网络的年耗电量与一些中等国家相当。批评者认为这是巨大的资源浪费，支持者则认为能源消耗换来了最安全的去中心化网络。PoS 的能耗极低，比 PoW 降低约 99.95%，这是以太坊转向 PoS 的核心动因之一。

# 安全性对比

PoW 的安全性来自物理资源的沉没成本和电力消耗，攻击一次需要持续投入巨额电费。PoS 的安全性来自经济惩罚——攻击者的质押金会被销毁，攻击成本是资本性的。两种模型的攻击经济模型不同，各有支持者和批评者。

# 去中心化考量

PoW 中算力集中在拥有廉价电力和高效矿机的实体手中，存在矿池中心化的问题。PoS 中质押集中度也是一大关注点，因为质押门槛（32 ETH）和流动性质押衍生品（LSD）可能导致少数实体控制大量质押份额。
